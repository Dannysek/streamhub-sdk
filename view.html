Class: streamhub-sdk/view

streamhub-sdk/view

new streamhub-sdk/view(opts)

Defines a base view object that can be bound to any number of stream-managers. Content is passed into a view via "add" event on "this". Subclasses are responsible for listening to the "add" events and using them to display streamed content.

<static> DEFAULT_REGISTRY

The default registry for Content -> ContentView rendering. Expects entries to always contain a "type" property, and either a view property (the type function itself) or a viewFunction property (a function that returns a type function, useful for conditional view selection.).
