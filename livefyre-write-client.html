<static> postContent(opts, callback)

Posts a piece of content to a Livefyre collection.

Parameters:
Name Type Description
opts Object

The livefyre collection options.

Properties
Name Type Description
network string

The name of the network in the livefyre platform
collectionId string

The livefyre collectionId for the conversation
lftoken string

The livefyre user auth token
body string

The content's body html with the following allowed tags: a, img, span, label, p, br, strong, em, u, blockquote, ul, li, ol, pre
media array

An Array of oEmbed JSON Objects to attach to the posted Content
callback function

A callback that is called upon success/failure of the write request. Callback signature is "function(error, data)".
